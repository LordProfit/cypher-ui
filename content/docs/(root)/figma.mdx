## Paid

- [cypher-ui/ui kit](https://cypher-uidesign.com) by [ Matt Wierzbicki](https://x.com/matsugfx) - A premium, always up-to-date UI kit for Figma - cypher-ui/ui compatible and optimized for smooth design-to-dev handoff.

## Free

- [cypher-ui/ui design system](https://www.figma.com/community/file/1203061493325953101) by [Pietro Schirano](https://twitter.com/skirano) - A design companion for cypher-ui/ui. Each component was painstakingly crafted to perfectly match the code implementation.
- [Obra cypher-ui/ui](https://www.figma.com/community/file/1514746685758799870/obra-cypher-ui-ui) by [Obra Studio](https://obra.studio/) - Carefully crafted kit designed in the philosophy of cypher-ui, tracks v4, MIT licensed
